# ⏰ Reloj-php

Este proyecto es una aplicación web simple construida con **PHP**, **HTML**, y configuraciones usando **Docker** y **NGINX**, cuyo propósito principal es mostrar un reloj digital funcional como práctica de desarrollo web.


## 🚀 Tecnologías utilizadas

- **PHP** — Backend del reloj
- **HTML / CSS / JS** — Interfaz del reloj
- **Docker / Docker Compose** — Entorno de desarrollo y despliegue
- **NGINX** — Servidor web para servir el proyecto

## 🛠️ Requisitos

- Docker
- Docker Compose
